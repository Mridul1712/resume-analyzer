# 🚀 Resume Analyzer

A web-based application that analyzes resumes and compares them with job descriptions to provide skill insights and match scores.

---

## 🔍 Features

- 📄 Upload PDF resume
- 🧠 Extract skills from resume
- 📊 Calculate job match score
- ❌ Identify missing skills
- 🎨 Interactive UI with animations

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML, CSS
- PyPDF2

---

## ⚙️ How It Works

1. Upload your resume (PDF format)
2. Paste a job description
3. The system:
   - Extracts skills from resume
   - Matches them with job requirements
   - Calculates a match percentage
   - Displays missing skills

---
